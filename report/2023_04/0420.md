# 学んだこと
分割したソースコードを読み込む手段の違い
- require：読み込み１回
- require_relative：読み込み１回
- load：読み込み何度でも
*ほかに、パスの参照方法、拡張子の省略の有無も違う

クラス分けをしたあとに、下記をGameクラスに記述するとよさそう。

```Ruby
require_relative 'deck'
require_relative 'hand'
require_relative 'player'
require_relative 'dealer'
```
